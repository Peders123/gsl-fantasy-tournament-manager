Welcome to TSL Mercury

The Mercury API is a powerful and flexible API designed to support Tanuki Smite League. Our API allows you to access a variety of endpoints that provide data and functionality essential for managing and participating in the league.

More features for support of the league will be coming very soon.

Endpoints:

All endpoints support standard operations: GET, POST, DELETE, PUT, OPTIONS

  • api-authentication
  • api/users
  • api/tournaments
  • api/players
  • api/captains

Getting Started

Currently the API is not open for public use. If you would like to get involved, please contact the owner of Tanuki Smite League.
